# Huevo D'Oro - Sistema Web de Avicultura

Sistema web desarrollado con React, PHP y MySQL para la gestión y promoción de productos avícolas.

## Características

- **Panel de Administración**: Gestión completa de productos, recetas, galerías, proveedores y más
- **Vista de Cliente**: Catálogo de productos, recetas, galería y contacto
- **Sistema de Autenticación**: Login y registro de usuarios
- **Gestión de Contenido**: Banners, imágenes y contenido dinámico

## Tecnologías

- **Frontend**: React + Vite
- **Backend**: PHP
- **Base de Datos**: MySQL
- **Estilos**: CSS personalizado

## Instalación

1. Instalar dependencias:
```bash
npm install
```

2. Iniciar servidor de desarrollo:
```bash
npm run dev
```

3. Configurar base de datos MySQL (ver `backend/database.sql`)

4. Configurar XAMPP para PHP backend

## Estructura del Proyecto

```
huevo-d-oro/
├── src/              # Código fuente React
├── backend/          # API PHP
├── public/           # Archivos estáticos
└── uploads/          # Imágenes subidas
```
